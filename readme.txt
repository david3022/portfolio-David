# Portfolio-David

Este repositorio contiene el código fuente de Portfolio-David, una aplicación web sencilla.

## Instrucciones de Uso
Sigue estos pasos para ejecutar Portfolio-David en tu navegador web:

Opcion A: Directo desde el navegador
1. Descargar este repositorio 
2. Abrir cualquier navegador web y escribir la direccion donde tenemos guardado el archivo;si por ejemplo está en el escritorio debemos colocar la dirección: file:///C:/Users/User/Desktop/portfolio-Davic/index.html#inicio


Opcion B:A través de VS Code

1. Abre Visual Studio Code (VS Code) si no lo tienes abierto.

2. Abre la carpeta del proyecto Portfolio-David en VS Code.

3. En el explorador de archivos de VS Code, ubica el archivo HTML principal de tu proyecto (por ejemplo, `index.html`).

4. Haz clic con el botón derecho en el archivo HTML.

5. En el menú contextual que aparece, selecciona la opción "Abrir en el navegador predeterminado" (o similar).

6. Esto abrirá tu proyecto Portfolio-David en el navegador web que tengas configurado como predeterminado en tu sistema.

¡Eso es todo! Deberías poder ver tu proyecto web Portfolio-David en acción en tu navegador.

## Requisitos

Asegúrate de tener Visual Studio Code instalado, así como un navegador web (como Google Chrome o Mozilla Firefox) configurado como tu navegador predeterminado.
